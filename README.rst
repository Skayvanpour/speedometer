==========================================================
Simple GPS Speedometer
==========================================================

Jonathan P Dawson

2013

Requirements
============

You need the following packages to build the speedometer:

  + avr-gcc
  + avr-libc
  + uisp

Build
=====

Compile the source code:

> make

Program
=======

Set the AVR fuses:

> ./set_fuses

Program the AVR:

> make load
